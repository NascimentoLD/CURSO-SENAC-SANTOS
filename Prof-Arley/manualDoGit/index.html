Qual a diferença do GIT para o GITHUB

  • GIT

O git é a ferramenta onde trabalhamos tudo dentro do nosso repositorio Local.

  • GITHUB

O git é a ferramenta onde trabalhamos tudo dentro do nosso repositorio Remoto.

Comandos no GIT Bash

  • ls

    • Listar Sistema, todas as pastas e localidades da nossa maquina.

  • ctrl + L

    • Limpa a tela de códigos.

  • ctrl + ↑ (seta)

    • Faz com que apareça o histórico de comandos que já havíamos utilizado.

  • cd

    • Acessar o diretório (Acessar a Pasta).

  • cd ..

    • = Voltar para a pasta anterior.

  • cd ./

    • Adicionamos todo o caminho para onde queremos ir.

  • mkdir

    • Cria uma pasta

  • pwd

    • Nos mostra onde estamos da nossa maquina
