Funções Adicionais

  • git clone

      • Clonamos um repositório remoto, para um repositorio local ( ou seja, pegamos um repositorio do GITHUB para meu computador.) Assim conseguimos trabalhar em nosso projeto atraves de outra maquina (Outro computador), e depois de concluirmos podemos atualizar nosso repositorio para ter a versão mais recente

      • git fork

      O Git Fork faz tipo um "Clone" de um repositorio de alguem, para o seu proprio GitHub, porém ele não tras o repositorio para a sua maquina, somente mediante "git clone"
